# Sales Performance Dashboard - Power BI

## Project Overview
This project presents an interactive Power BI dashboard designed to monitor sales performance and profitability.

## Business Objectives
- Track Revenue and Profit
- Monitor YoY Growth and Margin
- Analyze performance by Fiscal Year, Country, Category and Channel
- Support data-driven decision making

## Key KPIs
- Revenue (YTD)
- Profit
- Margin (%)
- YoY Growth
- Mean Basket Revenue

## Tools & Technologies
- Power BI
- DAX
- Data modeling
- Time intelligence
